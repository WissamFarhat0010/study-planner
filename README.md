# Study Planner App

## Description
This is a full-stack Study Planner application built using React, Node.js, Express, and MongoDB Atlas. The application allows users to manage study tasks efficiently.

## Features
- Create, read, update, and delete tasks (CRUD)
- Filter tasks by status and priority
- View task statistics
- Responsive frontend built with React
- Backend API built with Express
- Database using MongoDB Atlas

## Technologies Used
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Version Control: Git & GitHub

## Setup Instructions

### 1. Clone the repository
git clone https://github.com/WissamFarhat0010/study-planner.git

cd study-planner

### 2. Install dependencies
npm install
cd client
npm install
cd ../server
npm install


### 3. Create .env file (inside /server folder)

MONGO_URI=your_mongodb_connection_string
PORT=5000


### 4. Run the application

npm run dev


### 5. Open in browser
Frontend:

http://localhost:5173


Backend API:

http://localhost:5000/api/tasks
